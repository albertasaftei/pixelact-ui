## Dialog

A pixel art styled dialog component built on top of shadcn/ui's Dialog component. Features retro styling with multiple variants and full support for all underlying dialog props.

### Installation

Add the Dialog component to your project using the shadcn CLI:

```bash
npx shadcn@latest add https://pixelactui.com/r/pixelact-dialog.json
```

This will automatically install the component and its dependencies to your project.
