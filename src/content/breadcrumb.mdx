## Breadcrumb

A pixel art styled breadcrumb component built on top of shadcn/ui's Breadcrumb. Features retro styling, pixel font, and full support for all underlying breadcrumb props. Useful for navigation and showing the user's current location in the app.

### Installation

Add the Breadcrumb component to your project using the shadcn CLI:

```bash
npx shadcn@latest add https://pixelactui.com/r/pixelact-breadcrumb.json
```

This will automatically install the component and its dependencies to your project.
