## Collapsible

A pixel art styled collapsible component built on top of shadcn/ui's Collapsible. Features retro styling, pixel font, and full support for all underlying collapsible props. Useful for hiding and showing content, FAQs, and more.

### Installation

Add the Collapsible component to your project using the shadcn CLI:

```bash
npx shadcn@latest add @pixelact-ui/collapsible
```

This will automatically install the component and its dependencies to your project.
