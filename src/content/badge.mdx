## Badge

A pixel art styled badge component built on top of shadcn/ui's Badge component. Features retro styling, pixel font, and multiple variants. Useful for status indicators, labels, and more.

### Installation

Add the Badge component to your project using the shadcn CLI:

```bash
npx shadcn@latest add @pixelact-ui/badge
```

This will automatically install the component and its dependencies to your project.
