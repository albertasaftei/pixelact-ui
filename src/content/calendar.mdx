## Calendar

A pixel art styled calendar component built on top of shadcn/ui's Calendar. Features retro styling, pixel font, and full support for all underlying calendar props. Useful for date picking, scheduling, and more.

### Installation

Add the Calendar component to your project using the shadcn CLI:

```bash
npx shadcn@latest add @pixelact-ui/calendar
```

This will automatically install the component and its dependencies to your project.
