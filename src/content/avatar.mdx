
import { Avatar, AvatarImage, AvatarFallback } from "@/components/ui/pixelact-ui/avatar";

## Avatar

A pixel art styled avatar component built on top of shadcn/ui's Avatar component. Features retro styling, pixel font, and full support for all underlying avatar props. Useful for user profile images, team members, and more.

### Installation

Add the Avatar component to your project using the shadcn CLI:

```bash
npx shadcn@latest add https://pixelactui.com/r/pixelact-avatar.json
```

This will automatically install the component and its dependencies to your project.
