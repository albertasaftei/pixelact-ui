## Input

A pixel art styled input component with retro styling and multiple variants. Features pixel font styling and full support for all HTML input attributes.

### Installation

Add the Input component to your project using the shadcn CLI:

```bash
npx shadcn@latest add https://pixelactui.com/r/pixelact-input.json
```

This will automatically install the component and its dependencies to your project.
