## Alert Dialog

A pixel art styled alert dialog component built on top of shadcn/ui's AlertDialog component. Features retro styling with multiple variants and full support for all underlying alert dialog props. Perfect for destructive actions, confirmations, and important user decisions.

### Installation

Add the Alert Dialog component to your project using the shadcn CLI:

```bash
npx shadcn@latest add @pixelact-ui/alert-dialog
```

This will automatically install the component and its dependencies to your project.

  return (
    <AlertDialog>

        <Button variant="destructive">Delete Account</Button>
      </AlertDialogTrigger>
      <AlertDialogContent>
        <AlertDialogHeader>
          <AlertDialogTitle>Are you absolutely sure?</AlertDialogTitle>
          <AlertDialogDescription>

          <AlertDialogCancel asChild>
            <Button variant="link">Cancel</Button>
          </AlertDialogCancel>
          <AlertDialogAction asChild>
            <Button variant="destructive">Yes, delete account</Button>
          </AlertDialogAction>
        </AlertDialogFooter>
      </AlertDialogContent>
    </AlertDialog>
  );
}`}
