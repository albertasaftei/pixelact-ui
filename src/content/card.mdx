## Card

A pixel art styled card component built on top of shadcn/ui's Card. Features retro styling, pixel font, and full support for all underlying card props. Useful for grouping content, displaying information, and more.

### Installation

Add the Card component to your project using the shadcn CLI:

```bash
npx shadcn@latest add @pixelact-ui/card
```

This will automatically install the component and its dependencies to your project.
