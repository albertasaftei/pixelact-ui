## Command

A pixel art styled command palette component built on top of shadcn/ui's Command. Features retro styling, pixel font, and full support for all underlying command props. Useful for search, navigation, and quick actions.

### Installation

Add the Command component to your project using the shadcn CLI:

```bash
npx shadcn@latest add @pixelact-ui/command
```

This will automatically install the component and its dependencies to your project.
