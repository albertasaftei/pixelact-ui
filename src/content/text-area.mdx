## Textarea

A pixel art styled textarea component built on top of shadcn/ui's Textarea component. Features retro styling with multiple variants and full support for all HTML textarea attributes.

### Installation

Add the Textarea component to your project using the shadcn CLI:

```bash
npx shadcn@latest add https://pixelactui.com/r/pixelact-textarea.json
```

This will automatically install the component and its dependencies to your project.
