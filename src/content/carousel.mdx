## Carousel

A pixel art styled carousel component built on top of embla-carousel and shadcn/ui's Carousel. Features retro styling, pixel font, and full support for all underlying carousel props. Useful for image sliders, product showcases, and more.

### Installation

Add the Carousel component to your project using the shadcn CLI:

```bash
npx shadcn@latest add https://pixelactui.com/r/pixelact-carousel.json
```

This will automatically install the component and its dependencies to your project.
