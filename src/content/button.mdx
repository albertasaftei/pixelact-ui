## Button

A pixel art styled button component built on top of shadcn/ui's Button component. Features retro styling with multiple variants and full support for all underlying button props.

### Installation

Add the Button component to your project using the shadcn CLI:


```bash
npx shadcn@latest add https://pixelactui.com/r/pixelact-button.json
```

This will automatically install the component and its dependencies to your project.
