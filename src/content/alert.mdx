import { Alert, AlertTitle, AlertDescription } from "@/components/ui/pixelact-ui/alert";

## Alert

A pixel art styled alert component built on top of shadcn/ui's Alert component. Features retro styling with multiple variants and full support for all underlying alert props.

### Installation

Add the Alert component to your project using the shadcn CLI:

```bash
npx shadcn@latest add https://pixelactui.com/r/pixelact-alert.json
```

This will automatically install the component and its dependencies to your project.
