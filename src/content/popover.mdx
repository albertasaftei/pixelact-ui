## Popover

A pixel art styled popover component built on top of shadcn/ui's Popover. Features retro styling, pixel font, and full support for all underlying popover props. Useful for tooltips, menus, and more.

### Installation

Add the Popover component to your project using the shadcn CLI:

```bash
npx shadcn@latest add https://pixelactui.com/r/pixelact-popover.json
```

This will automatically install the component and its dependencies to your project.
