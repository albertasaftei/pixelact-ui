## Hover Card

A pixel art styled hover card component built on top of Radix UI's HoverCard. Features retro styling, pixel font, and full support for all underlying hover card props. Useful for showing additional information on hover, user previews, and more.

### Installation

Add the HoverCard component to your project using the shadcn CLI:

```bash
npx shadcn@latest add https://pixelactui.com/r/pixelact-hover-card.json
```

This will automatically install the component and its dependencies to your project.
