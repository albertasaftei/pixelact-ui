## Tooltip

A pixel art styled tooltip component built on top of shadcn/ui's Tooltip. Features retro styling, pixel font, and full support for all underlying tooltip props. Useful for hints, labels, and additional info on hover or focus.

### Installation

Add the Tooltip component to your project using the shadcn CLI:

```bash
npx shadcn@latest add https://pixelactui.com/r/pixelact-tooltip.json
```

This will automatically install the component and its dependencies to your project.
