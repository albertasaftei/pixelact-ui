## Label

A pixel art styled label component built on top of shadcn/ui's Label component. Features retro pixel font styling and full support for all HTML label attributes and accessibility features.

### Installation

Add the Label component to your project using the shadcn CLI:

```bash
npx shadcn@latest add @pixelact-ui/label
```

This will automatically install the component and its dependencies to your project.

### Label with Input

<CodePreview 
  language="tsx"
  code={`import { Label } from "@/components/ui/pixelact-ui/label";
import { Input } from "@/components/ui/pixelact-ui/input";

export function LabelWithInput() {
  return (
    <div style={{ display: "flex", flexDirection: "column", gap: "4px" }}>
      <Label htmlFor="username">Username</Label>
      <Input 
        id="username"
        type="text"
        placeholder="Enter your username"
      />
    </div>
  );
}`}
>
  <div style={{ display: "flex", flexDirection: "column", gap: "4px" }}>
    <Label htmlFor="username">Username</Label>
    <Input 
      className="w-full md:w-auto"
      id="username"
      type="text"
      placeholder="Enter your username"
    />
  </div>
</CodePreview>
