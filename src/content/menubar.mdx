              ## Menubar

              A pixel art styled menubar component built on top of shadcn/ui's Menubar. Features retro styling, pixel font, and full support for all underlying menubar props. Useful for app menus, navigation, and grouped actions.

              ### Installation

              Add the Menubar component to your project using the shadcn CLI:

              ```bash
              npx shadcn@latest add https://pixelactui.com/r/pixelact-menubar.json
              ```

              This will automatically install the component and its dependencies to your project.
