## Checkbox

A pixel art styled checkbox component built on top of shadcn/ui's Checkbox. Features retro styling, pixel font, and full support for all underlying checkbox props. Useful for forms, settings, and toggles.

### Installation

Add the Checkbox component to your project using the shadcn CLI:

```bash
npx shadcn@latest add @pixelact-ui/checkbox
```

This will automatically install the component and its dependencies to your project.
